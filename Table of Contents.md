# AWS Hands-on Labs Documentation

> **Overview**
>
> This repository contains a structured collection of hands-on AWS laboratory exercises designed to build practical skills across core AWS services, Infrastructure as Code (IaC), containerization, DevOps, networking, security, monitoring, and serverless computing.

---

# Table of Contents

| Lab        | Topic                                       | Coverage                                   |
| ---------- | ------------------------------------------- | ------------------------------------------ |
| **LAB 1**  | **AWS IAM & Security Setup**                | Roles, Policies, MFA, Access Keys          |
| **LAB 2**  | **VPC & Networking**                        | VPC, Subnets, IGW, NAT, SG, NACL           |
| **LAB 3**  | **EC2 Instance Management**                 | Launch, Key Pairs, EBS, AMI, ASG           |
| **LAB 4**  | **S3 Storage & Lifecycle**                  | Buckets, Versioning, Policies, Replication |
| **LAB 5**  | **CloudWatch Monitoring**                   | Dashboards, Alarms, Logs, Events           |
| **LAB 6**  | **CloudFormation (Infrastructure as Code)** | Templates, Stacks, Change Sets, SAM        |
| **LAB 7**  | **Terraform (Infrastructure as Code)**      | Init, Plan, Apply, State, Modules          |
| **LAB 8**  | **AWS CI/CD Pipeline**                      | CodeCommit, CodeBuild, CodePipeline        |
| **LAB 9**  | **Docker & Amazon ECR**                     | Images, Containers, Push to ECR            |
| **LAB 10** | **Amazon ECS Fargate**                      | Task Definitions, Services, Load Balancer  |
| **LAB 11** | **Amazon EKS (Kubernetes)**                 | Cluster, Node Groups, Deployments, Helm    |
| **LAB 12** | **AWS Lambda & Amazon EventBridge**         | Functions, Triggers, Rules, Serverless     |
| **LAB 13** | **Amazon RDS & Backup Services**            | MySQL, Snapshots, Restore, Backup Vault    |
| **LAB 14** | **Amazon Route 53 & AWS Systems Manager**   | DNS, Health Checks, SSM, Parameter Store   |

---

> **Note**
>
> Each lab is designed to be completed independently while progressively building AWS administration, DevOps, cloud infrastructure, automation, monitoring, and operational management skills.
